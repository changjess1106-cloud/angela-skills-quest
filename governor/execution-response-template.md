# Execution Response Templates - Updated for v2.1

## 成功範本

1. **CREATED FILES**
   - 列出新增檔案完整路徑。
2. **MODIFIED FILES**
   - 列出修改檔案完整路徑。
3. **RAW EVIDENCE**
   - 原始驗證結果：列出 Test-Path / 檔案內容 / 腳本輸出等詳細內容。
4. **VALIDATION RESULT**
   - 檢測結果：CLEAR PASS / PARTIAL / FAIL，對應原始證據。
5. **FINAL JUDGEMENT**
   - 判定結論：不得比原始證據更樂觀！

## 部分完成 / 阻塞範本

1. **COMPLETED PART**
   - 列出已完成部分清單。
2. **RAW EVIDENCE**
   - 原始驗證結果詳細內容。
3. **VALIDATION RESULT**
   - 清楚具體的問題點與成功部分。
4. **BLOCKER**
   - 明確描述阻塞原因。
5. **NEXT REQUIRED INPUT**
   - 需要的後續輸入或資訊。
