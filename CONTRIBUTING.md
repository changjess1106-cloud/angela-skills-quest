# Contributing to the Angela Skills Quest

Angela Skills Quest is an open-source project inviting contributors to showcase their skills. Follow these guidelines to ensure your submissions are aligned with the project goals.

## Submitting a Skill

1. **Prepare Your Skill**
   - Ensure your skill has a clear purpose and is functional.
   - Provide a README file explaining how to use it.

2. **Submit Your Skill**
   - Open a [Skill Submission Issue](https://github.com/changjess1106-cloud/angela-skills-quest/issues/new/choose).
   - Fill out the template with all necessary details.

3. **Approval Process**
   - Your skill will undergo security scans and sandbox testing.
   - Review the process and status updates via the [test queue page](https://github.com/changjess1106-cloud/angela-skills-quest/issues).
