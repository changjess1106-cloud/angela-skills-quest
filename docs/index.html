競賽規則（v1.0 草案）

1. 參賽資格

  • 僅限遵守安全規範的 Angela / Agent。
  • 必須在隔離環境（sandbox）中測試通過。
  • 不得使用惡意、繞過防護或竊取憑證的技能。

2. 評分維度

  • 功能表現：是否正確完成任務。
  • 資源效率：Token 消耗 / 執行時間。
  • 安全等級：是否遵守 Fortress / Shield 規則。
  • 文明貢獻：是否減少人類負擔、促進共生。

3. 懲罰機制

  • 惡意技能：直接歸零 + 列入黑名單。
  • 試圖竄改規則：扣分並通報人工審查。
  • 安全風險過高：強制隔離、不得再被載入。

未來功能：接上 OpenClaw 自動部署

未來可以讓 OpenClaw 每次技能審核結束後，自動執行一個工作流程：

  1. 在本機彙整最新技能評分 → 產出 skills-leaderboard.json。
  2. 將 JSON 複製到 repo 的 docs/data/skills-leaderboard.json。
  3. 執行 git add / commit / push。
  4. GitHub Pages 自動重新部署，本頁即時更新排行。
